# 🗺️ 네이버 지도에서 좌표 찾는 방법

## 방법 1: 네이버 지도 URL에서 좌표 추출

### ⚠️ 중요: URL 형식 확인

네이버 지도 URL에는 두 가지 형식이 있습니다:

#### 형식 A: 일반 URL (사용 가능) ✅
```
https://map.naver.com/...?c=경도,위도,줌레벨,0,0,0,dh
```
- `c=` 뒤의 첫 번째: **경도(lng)**
- `c=` 뒤의 두 번째: **위도(lat)**
- 바로 사용 가능!

#### 형식 B: UTMK 좌표 URL (변환 필요) ⚠️
```
https://map.naver.com/.../coordinates/14372390.2044643,4182808.8222222,...?c=19.00,0,0,0,dh
```
- `coordinates/` 뒤의 숫자들은 **UTMK 좌표계** (큰 숫자)
- 직접 사용 불가, 변환 필요
- **해결**: 장소를 클릭하여 일반 URL 형식으로 변경하거나, 구글 지도 사용

### 단계별 가이드

1. **네이버 지도에서 장소 검색**
   - https://map.naver.com 접속
   - 원하는 장소 검색 (예: "남천해변시장")

2. **장소 클릭하여 상세 페이지로 이동**
   - UTMK 좌표 URL이 나오면, 장소를 클릭하면 일반 URL로 변경됨

3. **주소창의 URL 확인**
   - 일반 URL 형식: `?c=경도,위도,줌레벨,0,0,0,dh`
   - UTMK 형식이면 → 구글 지도 사용 권장

4. **좌표 추출**
   - `c=` 뒤의 첫 번째 숫자: **경도(lng)**
   - `c=` 뒤의 두 번째 숫자: **위도(lat)**
   - ⚠️ **주의**: 네이버 지도는 경도가 먼저, 위도가 나중에 나옵니다!

### 예시

URL이 다음과 같다면:
```
https://map.naver.com/...?c=129.1108,35.1556,15,0,0,0,dh
```

- **경도(lng)**: 129.1108
- **위도(lat)**: 35.1556

### data.js에 입력할 때

```javascript
lat: 35.1556,  // 위도 (두 번째 숫자)
lng: 129.1108, // 경도 (첫 번째 숫자)
```

## 방법 2: 네이버 지도에서 직접 확인

1. **네이버 지도 접속**
2. **장소 검색**
3. **지도에서 해당 위치 클릭**
4. **우측 정보창에서 주소 확인**
5. **주소를 복사하여 다른 방법으로 좌표 찾기**

## 방법 3: 구글 지도 사용

1. **구글 지도 접속** (https://maps.google.com)
2. **장소 검색**
3. **위치에서 우클릭**
4. **"좌표 복사" 클릭**
5. **형식**: `35.1556, 129.1108` (위도, 경도)
   - 첫 번째: 위도(lat)
   - 두 번째: 경도(lng)

## 방법 4: 좌표 변환 사이트 사용

- **네이버 좌표계 변환**: 네이버 지도 API 좌표 변환 도구 사용
- **구글 지도 좌표**: 구글 지도에서 찾은 좌표를 그대로 사용 가능

## ⚠️ 중요 사항

### 좌표 순서 주의!

- **네이버 지도 URL**: `c=경도,위도` (경도가 먼저!)
- **구글 지도**: `위도,경도` (위도가 먼저!)
- **data.js**: `lat: 위도, lng: 경도` (위도가 먼저!)

### 한국 지역 대략 좌표

- **서울**: lat: 37.4~37.6, lng: 126.9~127.1
- **부산**: lat: 35.1~35.2, lng: 129.0~129.2
- **대구**: lat: 35.8~35.9, lng: 128.5~128.7
- **인천**: lat: 37.4~37.5, lng: 126.6~126.7

### 좌표 범위 확인

좌표가 이상하면:
- 한국 위도: 약 33~38
- 한국 경도: 약 124~132
- 이 범위를 벗어나면 잘못된 좌표일 가능성이 높습니다!

## 📝 예시: 남천해변시장 좌표

네이버 지도에서 확인한 부산 남천해변시장 좌표:
- **위도(lat)**: 35.1556
- **경도(lng)**: 129.1108

```javascript
{
    lat: 35.1556,
    lng: 129.1108
}
```

## 💡 팁

- 좌표는 소수점 4자리 정도면 충분합니다
- 정확한 좌표보다는 대략적인 위치도 괜찮습니다
- 지도에서 마커 위치를 확인하고 필요하면 조정하세요
