# 붕어빵 가게 정보 서비스 기획서

## 📱 프로젝트 개요

### 프로젝트명
**붕세권** - 붕어빵 가게 정보 제공 모바일 웹앱

### 목적
사용자가 주변 또는 원하는 지역의 붕어빵 가게를 쉽게 찾고, 가게 정보를 확인할 수 있는 모바일 웹앱 서비스

### 타겟 사용자
- 붕어빵을 좋아하는 일반 사용자
- 새로운 붕어빵 가게를 찾고 싶은 사람
- 주변 붕어빵 가게 정보가 필요한 사람

---

## 🎯 핵심 기능

### 1. 지도 기반 가게 찾기
- **네이버 지도 API** 활용
- 지도에 붕어빵 가게 위치를 마커로 표시
- 지도 이동/확대/축소로 다양한 지역 탐색
- 현재 위치 기반 주변 가게 찾기

### 2. 가게 목록 보기
- 리스트 형태로 가게 정보 표시
- 가게 이름, 주소, 거리 정보
- 간단한 가게 이미지 또는 아이콘

### 3. 가게 상세 정보
- 가게 이름, 주소, 전화번호
- 영업 시간
- 메뉴 및 가격 정보
- 가게 사진
- 지도에서 위치 확인

### 4. 검색 기능
- 지역명으로 검색 (예: "강남구", "홍대")
- 가게 이름으로 검색
- 현재 위치 기반 주변 검색

### 5. 필터 기능 (선택)
- 영업 중인 가게만 보기
- 거리순 정렬
- 평점순 정렬 (추후 리뷰 기능 추가 시)

---

## 🛠 기술 스택

### 프론트엔드
- **HTML5**: 기본 구조
- **CSS3**: 스타일링 및 반응형 디자인
- **JavaScript (Vanilla)**: 기능 구현
- **네이버 지도 API**: 지도 및 위치 서비스

### 데이터 관리
- **초기**: JavaScript 객체 또는 JSON 파일로 정적 데이터 관리
- **추후 확장**: 백엔드 API 연동 가능 (Node.js, Express 등)

### 모바일 최적화
- 반응형 웹 디자인 (Mobile First)
- 터치 친화적 UI
- PWA (Progressive Web App) 고려 (추후)
