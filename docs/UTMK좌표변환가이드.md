# 🔄 네이버 지도 UTMK 좌표 변환 가이드

## 문제 상황

제공하신 URL 형식:
```
https://map.naver.com/p/search/.../coordinates/14372390.2044643,4182808.8222222,...?c=19.00,0,0,0,dh
```

이 URL의 `coordinates/14372390.2044643,4182808.8222222`는 **UTMK 좌표계**입니다.
- UTMK는 네이버 지도 내부에서 사용하는 투영 좌표계
- 우리가 사용하는 `lat`, `lng`는 **WGS84 좌표계** (위도/경도)
- **직접 변환이 필요합니다!**

## ✅ 해결 방법

### 방법 1: 네이버 지도에서 일반 URL 형식으로 확인 (가장 쉬움) ⭐

1. **네이버 지도에서 해당 장소 검색**
2. **장소를 클릭하여 상세 페이지로 이동**
3. **일반적인 URL 형식 확인**
   - 형식: `?c=경도,위도,줌레벨,...`
   - 예: `?c=129.1108,35.1556,15,0,0,0,dh`
   - 첫 번째: **경도(lng)**: 129.1108
   - 두 번째: **위도(lat)**: 35.1556

### 방법 2: 구글 지도 사용 (추천) ⭐⭐⭐

1. **구글 지도 접속** (https://maps.google.com)
2. **주소 검색**: "부산 수영구 광안해변로15번길 77" 또는 "남천해변시장"
3. **위치에서 우클릭**
4. **"좌표 복사" 클릭**
5. **형식**: `35.1556, 129.1108`
   - 첫 번째: **위도(lat)**: 35.1556
   - 두 번째: **경도(lng)**: 129.1108

### 방법 3: 네이버 지도에서 우클릭

1. **네이버 지도에서 해당 위치로 이동**
2. **지도에서 우클릭**
3. **"이 위치의 주소" 또는 좌표 확인**
4. **일반적으로 위도/경도 형식으로 표시됨**

### 방법 4: 좌표 변환 도구 사용

네이버 지도 API의 좌표 변환 기능을 사용하거나, 온라인 좌표 변환 사이트를 이용할 수 있습니다.

## 📝 남천해변시장 좌표 (참고)

부산 남천해변시장의 대략적인 좌표:
- **위도(lat)**: 35.1556
- **경도(lng)**: 129.1108

```javascript
{
    lat: 35.1556,
    lng: 129.1108
}
```

## ⚠️ 중요 사항

### UTMK vs WGS84

- **UTMK 좌표**: `14372390.2044643, 4182808.8222222` (큰 숫자)
  - 네이버 지도 내부 좌표계
  - 직접 사용 불가
  - 변환 필요

- **WGS84 좌표**: `35.1556, 129.1108` (작은 숫자)
  - 위도/경도 (일반적으로 사용)
  - 우리가 사용하는 형식
  - 바로 사용 가능

### 좌표 범위 확인

- **한국 위도**: 약 33~38
- **한국 경도**: 약 124~132
- 이 범위를 벗어나면 잘못된 좌표일 수 있습니다!

## 💡 추천 방법

**가장 쉬운 방법**: 구글 지도 사용
1. 구글 지도에서 주소 검색
2. 우클릭 → 좌표 복사
3. 바로 사용 가능!

## 🔧 UTMK → WGS84 변환 (고급)

프로그래밍으로 변환하려면 네이버 지도 API의 좌표 변환 기능을 사용해야 합니다.
하지만 일반적으로는 위의 방법 1, 2를 사용하는 것이 더 쉽습니다.
