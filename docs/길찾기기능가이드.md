# 🚗 네이버 지도 길찾기 연동 가이드

## 구현된 기능

가게 상세 정보 화면에 **"길찾기" 버튼**이 추가되었습니다.

### 작동 방식

1. **현재 위치가 있는 경우**
   - 출발지: 현재 위치
   - 도착지: 선택한 가게
   - 네이버 지도에서 현재 위치에서 가게까지의 경로 표시

2. **현재 위치가 없는 경우**
   - 출발지: 사용자가 네이버 지도에서 직접 입력
   - 도착지: 선택한 가게
   - 네이버 지도에서 출발지를 입력하면 경로 표시

## 사용 방법

1. **가게 선택**
   - 지도에서 마커 클릭 또는 목록에서 가게 클릭

2. **가게 상세 정보 확인**
   - 모달에서 가게 정보 확인

3. **길찾기 버튼 클릭**
   - "🚗 길찾기" 버튼 클릭
   - 네이버 지도가 새 창에서 열림
   - 자동으로 길찾기 화면 표시

## 기술 구현

### 네이버 지도 길찾기 URL 형식

```
https://map.naver.com/v5/directions/{출발지}/{도착지}
```

- **출발지**: `위도,경도,이름` 또는 `-` (사용자 입력)
- **도착지**: `위도,경도,가게이름`

### 예시

```
https://map.naver.com/v5/directions/35.139216,129.109378,현재위치/35.155839,129.121462,삼바리 광안점 붕어빵
```

## 코드 위치

- **길찾기 함수**: `app.js`의 `openNavigation()` 함수
- **버튼 추가**: `app.js`의 `showStoreDetail()` 함수
- **스타일**: `styles.css`의 `.btn-navigation` 클래스

## 기능 개선 가능 사항

### 향후 추가 가능한 기능

1. **출발지 직접 입력**
   - 사용자가 출발지를 직접 입력할 수 있는 입력창 추가

2. **교통수단 선택**
   - 자동차, 대중교통, 도보 등 선택 가능

3. **지도 내 경로 표시**
   - 네이버 지도 API의 Directions API를 사용하여
   - 웹페이지 내에서 직접 경로 표시

4. **예상 시간 및 거리**
   - 경로 계산 후 예상 시간과 거리 표시

## 참고

- 네이버 지도 v5 API의 길찾기 URL을 사용
- 새 창에서 열리므로 사용자가 원래 페이지로 돌아올 수 있음
- 모바일에서는 네이버 지도 앱으로 자동 연결될 수 있음
