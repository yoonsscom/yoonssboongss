# 🏪 가게 추가 가이드

## 📋 가게 추가 방법

### 1단계: data.js 파일 열기

`data.js` 파일을 에디터에서 엽니다.

### 2단계: 마지막 가게 찾기

파일 맨 아래쪽에 있는 마지막 가게를 찾습니다 (현재는 id: 6).

### 3단계: 쉼표 추가

마지막 가게의 닫는 중괄호 `}` 뒤에 **쉼표(`,`)를 추가**합니다.

```javascript
    {
        id: 6,
        // ... 가게 정보 ...
    },  // ← 여기에 쉼표 추가!
```

### 4단계: 새 가게 추가

쉼표 뒤에 새 가게 정보를 추가합니다.

## 📝 가게 정보 작성 방법

### 필수 정보

1. **id**: 고유 번호 (기존 최대값 + 1)
   - 현재 최대: 6
   - 새 가게: 7, 8, 9... 순서대로

2. **name**: 가게 이름
   ```javascript
   name: "내가 좋아하는 붕어빵집"
   ```

3. **address**: 주소
   ```javascript
   address: "서울시 강동구 천호동 123-45"
   ```

4. **phone**: 전화번호
   ```javascript
   phone: "02-1234-5678"
   ```

5. **lat, lng**: 위도와 경도 (지도 위치)
   - 위도(lat): 37.xxxx (서울 기준 약 37.4~37.6)
   - 경도(lng): 126.xxxx 또는 127.xxxx (서울 기준 약 126.9~127.1)
   - **좌표 찾는 방법**은 아래 참고

6. **hours**: 영업시간
   ```javascript
   hours: {
       open: "10:00",   // 오픈 시간
       close: "22:00"   // 마감 시간
   }
   ```

7. **menu**: 메뉴 배열
   ```javascript
   menu: [
       { name: "붕어빵 1개", price: 1000 },
       { name: "붕어빵 3개", price: 2500 },
       { name: "슈크림 붕어빵", price: 1500 }
   ]
   ```

8. **image**: 가게 이미지 URL
   ```javascript
   image: "https://via.placeholder.com/400x200?text=가게이름"
   ```

## 🗺️ 좌표(lat, lng) 찾는 방법

### 방법 1: 네이버 지도에서 찾기

1. 네이버 지도(https://map.naver.com) 접속
2. 가게 주소 검색
3. 지도에서 가게 위치 클릭
4. 주소창의 URL에서 좌표 확인
   - 예: `https://map.naver.com/v5/search/.../place/...?c=127.1234,37.5678,15,0,0,0,dh`
   - `c=127.1234,37.5678` → lng: 127.1234, lat: 37.5678

### 방법 2: 구글 지도에서 찾기

1. 구글 지도(https://maps.google.com) 접속
2. 가게 주소 검색
3. 위치에서 우클릭 → 좌표 복사
4. 형식: `37.1234, 127.5678` → lat: 37.1234, lng: 127.5678

### 방법 3: 대략적인 좌표 사용

서울 지역 대략 좌표:
- 강남구: lat: 37.49~37.52, lng: 127.02~127.04
- 강동구: lat: 37.52~37.55, lng: 127.12~127.15
- 마포구: lat: 37.54~37.57, lng: 126.91~126.94
- 종로구: lat: 37.56~37.58, lng: 126.97~127.00

## 📝 완성 예시

```javascript
{
    id: 7,
    name: "천호동 붕어빵",
    address: "서울시 강동구 천호동 456-78",
    phone: "02-1234-5678",
    lat: 37.5384,
    lng: 127.1234,
    hours: {
        open: "09:00",
        close: "21:00"
    },
    menu: [
        { name: "붕어빵 1개", price: 1000 },
        { name: "붕어빵 3개", price: 2500 },
        { name: "붕어빵 5개", price: 4000 },
        { name: "슈크림 붕어빵", price: 1500 }
    ],
    image: "https://via.placeholder.com/400x200?text=천호동+붕어빵"
}
```

## ⚠️ 주의사항

1. **쉼표(`,`) 확인**
   - 마지막 가게가 아닌 경우: 쉼표 필요 `},`
   - 마지막 가게인 경우: 쉼표 불필요 `}`

2. **id 중복 금지**
   - 각 가게는 고유한 id를 가져야 합니다
   - 숫자만 사용 (1, 2, 3...)

3. **문법 확인**
   - 모든 문자열은 따옴표(`"`)로 감싸기
   - 숫자는 따옴표 없이
   - 중괄호 `{}`, 대괄호 `[]` 짝 맞추기

4. **저장 후 새로고침**
   - 파일 저장 후 브라우저에서 `F5` 또는 `Ctrl + Shift + R`로 새로고침

## ✅ 확인 방법

가게 추가 후:
1. 브라우저 새로고침
2. 지도에서 새 마커 확인
3. 목록에서 새 가게 확인
4. 가게 클릭하여 상세 정보 확인

## 💡 팁

- 여러 가게를 한 번에 추가할 수 있습니다
- 기존 가게를 복사해서 수정하면 빠릅니다
- 좌표는 대략적으로 넣어도 되지만, 정확할수록 좋습니다
