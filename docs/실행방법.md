# 🚀 붕세권 실행 방법

## 📋 준비사항

### 1. 네이버 지도 API 키 발급 (필수)

#### 단계별 가이드

1. **네이버 클라우드 플랫폼 접속**
   - 브라우저에서 https://www.ncloud.com/ 접속
   - 회원가입 또는 로그인

2. **Application 등록**
   - 왼쪽 메뉴에서 "AI·NAVER API" 클릭
   - "Application" 메뉴 클릭
   - 상단의 "Application 등록" 버튼 클릭

3. **정보 입력**
   - Application 이름: `붕세권` (원하는 이름)
   - 서비스 선택: **"Web Dynamic Map v3"** 체크
   - 나머지는 기본값으로 두고 "등록" 클릭

4. **Client ID 확인**
   - 등록 후 나타나는 Client ID를 복사
   - 예: `abc123def456ghi789...`

### 2. API 키 설정

`index.html` 파일을 열고 72번째 줄을 찾아주세요:

```html
<script type="text/javascript" src="https://oapi.map.naver.com/openapi/v3/maps.js?ncpClientId=YOUR_CLIENT_ID"></script>
```

`YOUR_CLIENT_ID`를 발급받은 Client ID로 변경하세요:

```html
<script type="text/javascript" src="https://oapi.map.naver.com/openapi/v3/maps.js?ncpClientId=abc123def456ghi789"></script>
```

## 🖥 실행 방법

### 방법 1: 간단한 방법 (파일 직접 열기)

1. `index.html` 파일을 더블클릭
2. 기본 브라우저로 열림
3. **단점**: 일부 기능이 제한될 수 있음

### 방법 2: 로컬 웹 서버 사용 (권장) ⭐

#### Python이 설치되어 있는 경우

1. **터미널/명령 프롬프트 열기**
   - Windows: `Win + R` → `cmd` 입력 → Enter
   - 또는 PowerShell 열기

2. **프로젝트 폴더로 이동**
   ```bash
   cd "C:\Users\User\Documents\CODING\붕세권"
   ```

3. **웹 서버 실행**
   ```bash
   python -m http.server 8000
   ```
   
   또는 Python 3인 경우:
   ```bash
   python3 -m http.server 8000
   ```

4. **브라우저에서 접속**
   - 브라우저 주소창에 입력: `http://localhost:8000`
   - 또는: `http://127.0.0.1:8000`

5. **서버 종료**
   - 터미널에서 `Ctrl + C` 누르기

#### Node.js가 설치되어 있는 경우

1. **터미널/명령 프롬프트 열기**

2. **프로젝트 폴더로 이동**
   ```bash
   cd "C:\Users\User\Documents\CODING\붕세권"
   ```

3. **웹 서버 실행**
   ```bash
   npx http-server -p 8000
   ```

4. **브라우저에서 접속**
   - `http://localhost:8000` 접속

### 방법 3: VS Code Live Server 사용

1. **VS Code에서 프로젝트 폴더 열기**

2. **Live Server 확장 설치**
   - 확장 프로그램에서 "Live Server" 검색 후 설치

3. **실행**
   - `index.html` 파일에서 우클릭
   - "Open with Live Server" 선택

## ✅ 실행 확인

정상적으로 실행되면:

1. **화면 상단**: "🍞 붕세권" 로고와 현재 위치 버튼
2. **검색창**: 지역명 또는 가게명 검색 가능
3. **지도 탭**: 서울 지역 지도와 붕어빵 가게 마커 표시
4. **목록 탭**: 가게 목록 보기

## 🔧 문제 해결

### 지도가 표시되지 않아요
- API 키가 올바르게 설정되었는지 확인
- 브라우저 콘솔(F12)에서 오류 메시지 확인
- 네이버 클라우드 플랫폼에서 Application이 활성화되어 있는지 확인

### "현재 위치" 버튼이 작동하지 않아요
- 브라우저에서 위치 권한을 허용해야 합니다
- HTTPS 또는 localhost에서만 작동합니다

### Python/Node.js가 없어요
- 방법 1(파일 직접 열기)을 사용하거나
- VS Code Live Server 확장을 사용하세요

## 📱 모바일에서 테스트하기

1. **로컬 서버 실행** (방법 2 사용)

2. **컴퓨터와 같은 Wi-Fi에 연결**

3. **컴퓨터의 IP 주소 확인**
   - Windows: `ipconfig` 명령어 실행
   - IPv4 주소 확인 (예: 192.168.0.10)

4. **모바일 브라우저에서 접속**
   - `http://192.168.0.10:8000` 입력
