# 🚀 무료로 실시간 배포하기

## 추천 무료 호스팅 서비스

### 1. Netlify (가장 추천) ⭐⭐⭐

**장점:**
- 무료
- 설정 간단
- 자동 배포 (Git 연동)
- HTTPS 자동 제공
- 커스텀 도메인 지원

**단계:**
1. https://www.netlify.com 접속 및 회원가입
2. "Add new site" → "Deploy manually" 클릭
3. 프로젝트 폴더를 드래그 앤 드롭
4. 배포 완료! URL 제공됨

**주의사항:**
- 네이버 지도 API의 Web 서비스 URL에 Netlify 도메인 등록 필요
- 예: `https://your-site.netlify.app`

## 가장 쉬운 방법: Netlify

### 상세 가이드

#### 1단계: 프로젝트 준비
- 모든 파일이 준비되어 있는지 확인
- `index.html`, `app.js`, `data.js`, `styles.css` 확인

#### 2단계: Netlify 가입 및 배포
1. **Netlify 접속**: https://www.netlify.com
2. **회원가입** (GitHub, Google, Email 중 선택)
3. **"Add new site"** 클릭
4. **"Deploy manually"** 선택
5. **프로젝트 폴더 드래그 앤 드롭**
   - 또는 "Browse to upload" 클릭하여 폴더 선택
6. **배포 완료!**
   - 자동으로 URL 생성됨
   - 예: `https://random-name-12345.netlify.app`

#### 3단계: 네이버 지도 API 설정
1. **네이버 클라우드 플랫폼 접속**
2. **Application 수정**
3. **Web 서비스 URL에 Netlify URL 추가**
   - 예: `https://your-site.netlify.app`
   - 또는 `https://*.netlify.app` (와일드카드)
4. **저장**

#### 4단계: 확인
- Netlify에서 제공한 URL로 접속
- 지도가 정상 작동하는지 확인

## 커스텀 도메인 설정 (선택사항)

### Netlify에서 커스텀 도메인 사용
1. Netlify 대시보드 → Site settings → Domain management
2. "Add custom domain" 클릭
3. 도메인 입력
4. DNS 설정 안내 따르기

---

## 자동 배포 설정 (Git 연동)

### GitHub와 연동하면 자동 배포 가능

1. **GitHub에 프로젝트 업로드**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/username/repo-name.git
   git push -u origin main
   ```

2. **Netlify에서 Git 연동**
   - "Add new site" → "Import an existing project"
   - GitHub 선택 및 Repository 선택
   - 자동 배포 설정 완료!

3. **이후 작업**
   - 코드 수정 후 GitHub에 push
   - Netlify가 자동으로 배포!

## 주의사항

### 네이버 지도 API
- 배포한 URL을 Web 서비스 URL에 등록해야 함
- 여러 URL 등록 가능 (로컬, 배포 URL 모두 등록)

### 파일 크기
- 무료 플랜은 파일 크기 제한이 있을 수 있음
- 현재 프로젝트는 작으므로 문제 없음

### 트래픽
- 무료 플랜은 월 트래픽 제한이 있을 수 있음
- 개인 프로젝트는 충분함

## 문제 해결

### 지도가 안 보여요
- 네이버 클라우드 플랫폼에서 배포한 URL 등록 확인
- 브라우저 콘솔(F12)에서 오류 확인

### 배포가 안 돼요
- 파일 이름 확인 (대소문자 구분)
- `index.html`이 루트에 있는지 확인
